# 📋 Task Manager

Projeto de um gerenciador de tarefas moderno inspirado em interfaces minimalistas e aplicações de produtividade.

## 🚀 Objetivo

Desenvolver uma aplicação de gerenciamento de tarefas para praticar conceitos de desenvolvimento Front-End, incluindo estruturação de páginas, estilização e futuramente interatividade com JavaScript.

## 🎨 Funcionalidades

- Sidebar de navegação
- Lista de tarefas
- Separação entre tarefas pendentes e concluídas
- Indicadores de quantidade de tarefas
- Interface moderna e responsiva
- Busca de tarefas
- Sistema de prioridades
- Organização visual por categorias

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- JavaScript (em desenvolvimento)

## 📚 Conceitos Praticados

- Estrutura semântica HTML
- Organização de componentes
- Metodologia de nomenclatura CSS
- Responsividade
- Layouts com Flexbox e Grid
- Manipulação do DOM (futuramente)
- Gerenciamento de tarefas com JavaScript
